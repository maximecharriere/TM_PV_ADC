# TM_PV_ADC

## Setup 

1. Clone the repo
1. Install Python
1. Setup and install the virtual environment in a Powershell with: 
    ```python
    py -m venv .venv
    .\.venv\Scripts\Activate.ps1
    py -m pip install --upgrade pip
    py -m pip install -r requirements.txt
    ```
1. Add a `.env` file in the root directory with the variable `PARENT_DATA_DIR="path/to/database/directory"`
